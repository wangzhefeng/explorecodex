# Model Selection

Pick models based on latency, depth, and budget.

## Decision Framework

- small edits and quick checks: faster/cheaper profile
- medium feature work: balanced profile
- complex architecture or debugging: strongest reasoning profile

## Team Recommendation Pattern

- set a balanced default in config
- escalate to stronger models only when needed
- document model-switch rules in AGENTS.md
