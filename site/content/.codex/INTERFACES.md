# Codex Interfaces

Codex can be used through different interfaces, each suited to a workflow.

## Interface Choices

- CLI: fastest path for hands-on repo work
- IDE: keep coding context and edits in one place
- Web: lightweight access without local setup
- Codex app: interactive operations with richer UI controls

## Suggested Learning Order

1. start with CLI quickstart
2. add IDE integration for daily development
3. use Web/App for review and async collaboration scenarios
