# Goal

Use this skill when a task has a known checklist and output format.

## Execution Steps

1. restate the requested outcome
2. inspect relevant files and constraints
3. perform changes in small verifiable increments
4. run checks and summarize results

## Output Contract

- list changed files
- summarize behavioral impact
- note validation commands and outcomes
- call out known risks or follow-up work
